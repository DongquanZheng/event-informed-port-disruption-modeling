# Processed Result Files

This folder is ignored by default because most processed files can be large.

Two small summary files are intentionally kept for lightweight reproduction of the final notebook:

- `multiscale_two_stage_default_comparison_2021_2025.csv`
- `multiscale_best_thresholds_2021_2025.csv`

These files allow `notebooks/final_multiscale_model_summary.ipynb` to reproduce the main result tables and figures without requiring users to download and process the full GDELT event data.
